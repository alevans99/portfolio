NC News

A social news aggregation, web content rating, and discussion website with a front-end written in React and a back-end using Node.js and Express, tested using Jest.

--Project Description

NC News has articles which are divided into topics. Each article has user curated ratings and can be up-voted or down-voted using the API. Users can also add comments about an article, as well as submitting their own articles and creating new topics.

--About the Project

NC News is the combination of two projects created during my training as a Junior Full-Stack Developer at Northcoders. The projects were split into two separate sprints, firstly the backend for the app was created to provide a RESTful API to allow the website functionality.

The second sprint added a responsive frontend to the project using React.

Beat Drop

A vertical scrolling video game with integrated social features and leaderboards. The project uses React and PhaserJS for the front-end, and Firebase Authentication and Firestore as a database.

--Project Description

In Beat Drop you have to dodge platforms, collect power ups, and avoid power downs to get a high score. You can keep track of your personal bests in the leaderboards and compare yourself to the all time records. Interact with other players and congratulate them on their high scores and achievements in the social feed and customise your game with a variety of levels and characters.

--About the Project

Beat Drop was the culmination of a final group project while studying at Northcoders. The project was completed over 8 working days, with the group following agile principles, using Kanban and daily standups to track progress as the project developed. The project was a really good opportunity to practice version control as a team as well as a good opportunity to familiarise ourselves with Firebase cloud services.

Worth Starting

A TV episode comparison website built using React for the main app and and Chart.js for data visualisation. Data is pulled from the TV Maze API using Axios

--Project Description

'Worth Starting' allows you to search for a TV series and view its episode ratings as a graph, allowing you to check that the series doesn't go downhill after the initial episodes. You can interact with the graph to choose the orientation (for ease of viewing on mobile) and select individual episodes to get more information.

--About the Project

Following my graduation from Northcoders, I really wanted to revisit the Data Visualisation module and refresh the skills we learned. This project was based on the result of a 2 day sprint, where we looked at linking data from APIs with Chart.js.
